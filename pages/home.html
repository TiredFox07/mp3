The Go programming language has slowly taken over the cloud's infrastructure.

I hope this project will serve as a gateway for those that have a familiarity with other programming languages to find out what is Go and why is Go used professionally. My project will show: why Go is becoming more popular, provide prior context as to how Go got here, show use cases in which Go is an excellent fit when developing software, and show the array of tools available to developers to make writing Go code more convenient.
